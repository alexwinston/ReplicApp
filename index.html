Always visible
ReplicApp windows by default are modal and sit atop your desktop unobstructed for easy access and viewing.
Resizable
ReplicApp windows are resizable so that you can scale that movie or video down you are watching so it is less obvious. Alternatively keep tabs on portions of windows without requiring the clutter at full scale.
Notifications
ReplicApp can alert you when a window hasn't changed for a specified duration -- allowing you to be alerted when that build is complete or output has stopped. Even better, you can be alerted when the window image changes by a certain percentage.
Ease of use
Easy activation through the menubar or keyboard shortcut. It mimics the screenshot shortcut by default in order to make it easier to remember.
Configuration
ReplicApp offers several configuration options. Refresh speed of the replicated content, notifications when content hasn't changed for a specified duration or changes a particular percent, pinnable and resizeable.
